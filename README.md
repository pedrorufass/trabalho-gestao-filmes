# Sistema de Gestão de Filmes 🎬

### Descrição
Este projeto é uma aplicação de gestão de catálogo cinematográfico que permite o registo de diretores, filmes e avaliações. O sistema foi desenvolvido para demonstrar competências em Bases de Dados e integração com Python.

### Tecnologias Utilizadas
* **Linguagem:** Python 3.x
* **Banco de Dados:** PostgreSQL
* **Biblioteca:** `psycopg2` 
* **Ambiente:** VS Code

### Estrutura do Repositório
* `/diagrama`: Diagrama Entidade-Relacionamento (DER).
* `/ddl`: Scripts de criação das tabelas (Data Definition Language).
* `/dml`: Scripts de inserção e manipulação de dados (Data Manipulation Language).
* `/dql`: Consultas SQL e Joins (Data Query Language).
* `/src`: Código-fonte da aplicação Python.
